Changelog

  • In addition to relaying unconfirmed transactions and new blocks, Bitcoin F# now serves the best chain getheaders, getblocks and getdata. It allows other peers to synchronize from it and has been tested using another instance of Bitcoin F# and Bitcoin core in every possible combination.
  • Bitcoin F# supports Bip 37. With the support of filtered blocks, filtered transactions and Merkle blocks, SPV clients synchronise much faster.
  • A function that writes a bootstrap file was added.
  • Watch-only wallet: Insert public key hashes into the table keys. UTXO that match these keys will be put into tx.
